# Beanstalk::Client::Rspec

Allows easy mocking of Beanstalk::Client

## Installation

Add this line to your application's Gemfile:

    gem 'beanstalk-client-rspec'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install beanstalk-client-rspec
